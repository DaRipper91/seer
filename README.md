# Scout Tool

A fuzzy finder and script management tool.

## Components

- `scout`: Fuzzy find and act on system scripts.
- `scout-refresh`: Re-scans the home directory for scripts and updates the inventory.

## Usage

1. Source the fish functions in your fish config.
2. Run `scout` to search for scripts.
3. Run `scout-refresh` to update the script inventory (`~/scripts_inventory.txt`).
